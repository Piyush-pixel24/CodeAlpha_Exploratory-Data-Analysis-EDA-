# ⚽ Football Match Data Analysis using Python

This project performs **Exploratory Data Analysis (EDA)** on football match data using **Python, Pandas, Matplotlib, and Seaborn**.  
The analysis focuses on team performance, goals, possession statistics, match trends, formations, and other key football insights through statistical analysis and visualizations.

---

## 📌 Features

- Data Cleaning & Preprocessing
- Statistical Analysis
- Correlation Analysis
- Team Performance Analysis
- Match Result Visualization
- Goals & Possession Analysis
- Heatmaps, Histograms, Pie Charts & Bar Graphs

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
